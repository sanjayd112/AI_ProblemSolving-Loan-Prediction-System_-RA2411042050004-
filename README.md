# 💰 Loan Approval Prediction System (AI Project)

## 📌 Problem Description
This project is an AI-based Loan Approval Prediction System that predicts whether a loan will be **Approved or Rejected** based on applicant details like income, credit score, employment status, and loan amount.

---

## 🤖 Algorithm Used
- Logistic Regression (Classification Algorithm)
- Supervised Machine Learning

---

## 📊 Input Features
- Income  
- Credit Score  
- Employment Status  
- Loan Amount  

---

## ⚙️ How It Works
- Data preprocessing (encoding + scaling)
- Model training using Logistic Regression
- Model saved using pickle
- Flask web app for user input
- Real-time prediction output

---

## 🚀 How to Run

### Install dependencies
pip install flask numpy pandas scikit-learn

### Train model
python model.py

### Run application
python app.py

### Open in browser
http://127.0.0.1:5000/
